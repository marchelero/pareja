# LOVEPLAY ❤️🔥

¡Bienvenido a **LOVEPLAY**! La app definitiva para romper el hielo, encender la chispa y pasar un rato inolvidable en pareja (o en una previa que se salga de control 😉).

## 🚀 ¿Qué es LOVEPLAY?

LOVEPLAY es una aplicación móvil diseñada exclusivamente para parejas y amigos con derechos. Cuenta con diferentes minijuegos interactivos que combinan diversión, preguntas incómodas y dinámicas picantes.

Introduce tu nombre y el de tu acompañante... ¡y que empiece el juego!

## 🎮 Modos de Juego Actuales

Actualmente puedes disfrutar de 3 modos principales:

*   **Preguntas 💬:** ¿Qué tanto se conocen? Preguntas aleatorias para descubrir secretos, confesiones y preferencias. ¡Prepárate para ser honesto!
*   **Ruleta 🎰:** ¡Deja que la suerte decida! Gira la ruleta y prepárate para cumplir castigos, hacer confesiones o ganar recompensas. (Incluye dinámicas de todo tipo).
*   **Chupitos 🥃:** El clásico juego de beber, pero llevado al siguiente nivel. Cumple el reto o tómate un chupito. ¡Ideal para empezar la noche!

## 🚀 Próximamente (En desarrollo)

Porque la diversión no para, estamos preparando nuevas secciones:

*   **Hot 🔥:** Sube la temperatura con dinámicas más íntimas y atrevidas.
*   **Retos ⚡:** Desafíos físicos y mentales. ¿Te atreves?
*   **Otros 🎲:** ¡Sorpresas que estamos cocinando para futuras actualizaciones!

## 🛠️ Para los curiosos (Devs)

Si eres desarrollador y quieres ver cómo está hecha la magia, aquí tienes los detalles:

*   **Tecnología:** Construido 100% en [Flutter](https://flutter.dev/) con Dart.
*   **UI/UX:** Diseño moderno con efectos "Glassmorphism" (cristal), fondos de neón interactivos y animaciones fluidas.
*   **Almacenamiento:** Uso de `shared_preferences` para guardar los nombres de los jugadores localmente.

### ¿Cómo correr el proyecto?

1. Asegúrate de tener el SDK de Flutter instalado (versión `>=3.5.2`).
2. Clona este repositorio:
   ```bash
   git clone https://github.com/marchelero/pareja.git
   ```
3. Instala las dependencias:
   ```bash
   flutter pub get
   ```
4. ¡Corre la app en tu emulador o dispositivo físico!
   ```bash
   flutter run
   ```
